CARRUSELL

es una presentación de diapositivas para ciclar una serie de contenido, construido con CSS 3D y un poco de JavaScript. Funciona con una serie de imágenes, texto o marcado personalizado.

En los navegadores donde se admite la API Visibilidad de página, el carrusel evitará deslizarse cuando la página web no sea visible para el usuario.

(como cuando la pestaña del navegador está inactiva, la ventana del navegador está minimizada, etc.)

Ademas se debe tener en cuenta que los carruseles anidados no son compatibles, y que los carruseles generalmente no cumplen con los estándares de accesibilidad.

Los carruseles no ajustan automáticamente las dimensiones de la diapositiva. por lo tanto, puede ser necesario utilizar recursos adicionales o estilos personalizados para ajustar el tamaño del contenido

Opciones

Las opciones se pueden pasar a través de atributos de datos o JavaScript. Para los atributos de datos, agregue el nombre de la opción a data- como en data-interval="".

Nombre Tipo Valor por defecto Descripción
interval number 5000 La cantidad de tiempo que de tiempo que tardara en cambiar automáticamente un elemento. Si es falso, el carrusel no se activará automáticamente.
keyboard boolean true Si el carrusel debería reaccionar a eventos de teclado.
pause string | null "hover" Si se establece en "hover", pone en pausa el ciclo del carrusel cuando el cursor esta sobre el carrusel y reanuda el ciclo del carrusel cuando el cursor deja de estar sobre. Si se establece en null, al pasar el mouse sobre el carrusel no se detendrá.
ride string false Reproduce automáticamente el carrusel después de que el usuario cicla manualmente el primer elemento.
wrap boolean true Si el carrusel debe ciclar continuamente o tiene largos periodos detenido.
