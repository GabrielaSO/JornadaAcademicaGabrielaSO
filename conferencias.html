INTELIGENCIA ARTIFICIAL

Como introducción, se explico que la inteligencia artificial(desde 1950) es una necesidad para la evolución robótica, cuyo objetivo es imitar el comportamiento humano, buscando aprender y razonar, para ejecutar ciertas tareas, pero para que un robot aprenda, se le tienen que dar ordenes así como proporcionarle información en espera de una respuesta.

Los temas que abarca la Inteligencia artificial son muchos, entre algunos de ellos:

  • Sistemas autónomos
  • Machine Learning
  • Deep Learning
  • Redes Neuronales
  • Patrones de reconocimiento

El M.C. Pablo Salazar, comento que "La 'IA' esta en muchas areas, pudiendo abarcar casi todas, algunas de las áreas mencionadas fueron la medicina principalmente, ingeniería, videojuegos, etc.
La IA tiene 2 pilares:

  • Algoritmos genéticos
  • Redes Neuronales

También se hablo sobre Deep Learning que es machine Learning aplicado a Big Data, de manera que estas grandes cantidades de datos, se clasifiquen, esto es de gran importancia en la inteligencia de negocios, la cual consiste en obtener información de utilidad clasificarla esto mediante un proceso llamado ETL (Extracción, transformación y carga).
Continuando con Machine Learning, el M.C. hablo sobre las 2 maneras en las que una "maquina" aprende:

  1. Entrenamiento supervisado, en el cual se entrena a un programa ingresando datos clasificados para que este razone por si mismo y realice una o varias tareas.
  2. Entrenamiento no supervisado, se refiere a que se le ingresan datos no clasificados a un programa y este mediante un modelo matemático los procesa, siendo un aprendizaje autónomo.

Al final, fue presentado un sistema llamado WATSON, el cual entiende el lenguaje natural y es capaz de responder preguntas.

Criptografía Post-cuantica

Lic. Diego Armando Retana García

17 - octubre - 2017

Hora: 18:00 - 20:00

CRIPTOGRAFÍA POST-CUANTICA

Previamente, el Lic.Diego Retana, nos hablo sobre un sistema de inteligencia artificial similar a WATSON, llamado Relativity 6.

Comenzando con el tema de criptografía, la cual se define como una ciencia que desarrolla técnicas matemáticas para cifrar información,
Se tienen 2 tipos principales:

  1. Criptografía simétrica: solo utiliza una llave para cifrar un mensaje, misma llave que servirá para descifrarlo, por lo tanto el emisor y el receptor deben de conocer dicha llave.
  2. Criptografía asimétrica: hay 2 llaves, una púbica y una privada.
  • Algoritmos:

    • RSA: un algoritmo que hace uso de 2 llaves (criptografía asimétrica), este algoritmo calcula una clave pudiendo ser de distintos tamaños, para calcular la clave, se utiliza una "función de una vía", esto quiere decir que la función puede ser calculada sencillamente, pero para deshacer este calculo y así descifrar la clave, tomaría bastantes años para que una computadora pudiera hacerlo.

    DISPOSITIVOS ROBÓTICOS Y SU INTEGRACIÓN EN APLICACIONES CON REALIDAD VIRTUAL

    Se define la realidad como una representación en 3D de la realidad o una representación falsa de la realidad que permite

    • Enviar información hacia en mundo real de manera visual, auditiva o táctil.
    • Recibir información del mundo real.
    • Interacción con usuarios en el mundo real.

    Se hablo sobre 3 temas de principal interes para la realidad virtual

    Representación
    Para la representación de un ambiente sintético en 3D es necesario el modelado de objetos y texturas, y aunque entre mas fino y detallado sea un objeto 3D sera mas pesado para la PC, se vera mas real; además es importante evitar el traslape de objetos en un ambiente virtual.

    Interacción
    Para la interacción con estos objetos 3D, es necesario contar con hardware básico como teclado, ratón o joystick o hardware más avanzado como kinect, interfaz aptica, dispositivos de captura de movimiento, cyber globe, cyber grasp, mouse 3D, etc.

    Selección
    Se utilizan algoritmos para le detección de colisiones, los cuales determinan la proximidad de un objeto, algunos de estos algoritmos son:
    RAPID
    COLLIDE

    También se habló de los tipos de comportamiento:

    1. Cinemático: es el menos real que se puede tener
    2. Dinámico: es el mas real, estudia el movimiento de los objetos a partir de las fuerzas que se ejercen sobre ellos, en este tipo de comportamiento se utilizan leyes de la física regularmente de newton.
