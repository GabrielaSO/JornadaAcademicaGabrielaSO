Ventanas modales

¿Que son?

Las ventanas Modales en Bootstrap, son unas ventanas ocultas que contienen la información a mostrar cuando las hacemos visibles mediante un enlace o botón.

Se abren dentro del mismo entorno de la web, ya que su código está definido dentro de la misma web que lo contiene, y lo único que hacemos como usuarios, es que sea visible o invisible mediante enlaces o botones.

Las ventanas modales apoyan una serie de casos de uso, desde notificaciones para el usuario, hasta contenido completamente personalizado con subcomponentes, tamaños y mas.

Ejemplo de una ventana modal



¿Cómo funciona?

Las ventanas modales estan construidas con HTML, CSS y JavaScript, y se posicionan sobre todo el documento, eliminando el desplazamiento sobre el contenido del <body>.

Al hacer clic en el fondo, la ventana modal se cerrará de manera automatica.

Bootstrap solo admite una ventana modal a la vez, por to tanto, es probable que se tenga problemas cuando se anida una ventana modal dentro de otra.

Las ventanas modales utilizan position: fixed, lo cual a veces puede ser un poco particular por de su representacion debido a que hay algunas advertencias con el uso de modales en dispositivos móviles.

sin embargo, siempre que sea posible, se debe colocar la ventana modal en una posicion superior para evitar posibles interferencias con otros elementos.

Por último, el atributo de autofocus HTML no tiene ningún efecto en modales. A continuación, se muestra cómo se puede lograr el mismo efecto con JavaScript personalizado.
